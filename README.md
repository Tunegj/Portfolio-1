# Portfolio - Tone Gjerde

A single-page portfolio website showcasing the works and projects of Tone Gjerde.

## Description

This portfolio website was created to present three rewuired Noroff projects, each revisited and improved as part of Portfolio 1.
The site is built as a simple SPA using vanilla JavaScript, HTML, and CSS, and includes links to live deos, GitHub repositories, and supporting documentation.

The portfolio features the following projects:

- Rainy Days (Cross Course Project)
- Community Science Museum (Semester Project)
- Square Eyes (JS1 Course Assignment)

Each project includes a brief description, thumbnail image, Github repository link, and live site link.

## Built with:

- HTML
- CSS
- JavaScript (Vanilla)
- GitHub Pages (deployment)

## Getting Started

### Installing:

1. Clone the repository: `git clone https://github.com/Tunegj/Portfolio-1.git`

### Running the Project:

1. Open the `index.html` file in your web browser or use Live Server in VS Code.

No dependencies are required to run this project.

Projects included:

1. Rainy Days
   Cross Course Project with improvements including dynamic product rendering from API.
